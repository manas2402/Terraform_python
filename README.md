# Terraform Python Udemy Exercises

This repository contains Terraform configuration files for various exercises from the Udemy course on Terraform.

## Exercises

- **exercise1/**: Basic instance deployment
- **exercise2/**: Instance with keypair and security group
- **exercise3/**: Advanced instance setup with variables
- **exercise4/**: Instance with user data script
- **exercise5/**: Similar to exercise4 with variables
- **exercise6/**: Using remote backend

**Note:** Do not commit .tfstate files or private keys. Use your own AWS credentials and keys.

## Usage

1. `cd exerciseX`
2. `terraform init`
3. `terraform plan`
4. `terraform apply`
